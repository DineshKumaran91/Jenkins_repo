Jenkins is a continuous integration tool that allows continuous developement,test and deployment of newly created codes

Other CI tools:
Bamboo,Buildbot,ApacheGump,TravisCI
Popular one is Jenkins

Features in Jenkins:
 - Easy installation
 - Easy configuration
 - Plugins (many plugins to increase functinality)
 - Extensible
 - Distributed (works across multiple environment)

Jenkins Pipeline:
Developement --> Code commit --> Build --> Test --> Release --> Deploy/Deliver --> Production
                 -------------------------Jenkins-----------------------------
